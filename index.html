About

The starterkit allows you to create and build awesome sites without worrying about anything else. The only thing you should worry about is your code. Just clone the repository run npm install gulp and you're ready to go!. For more information about gulp tasks and the project, go to the starterkit documentation.

Starterkit Generator

I've created a yeoman generator to use with the starterkit. If you want to use different technologies (add / remove) something that you don't like you should use generator-starterkit. The generator-starterkit is a yeoman generator that will ask you for the technologies that you want to use and will create dynamically a project folder with all the necessary files and customized files based on your choices. To use the starterkit generator simply go to generator-starterkit read the documentation, and install it.
